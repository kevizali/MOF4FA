# Public repository for "AI-assisted screening of MOFs for formaldehyde removal: Combining GCMC simulation and machine learning"

Contents
1. p1_GAIresults.xlsx (prompt 1) contains the list of MOFs used in removal of VOCs generated by the GAI models.
2. p2_GAIresults.xlsx (prompt 2) contains the list of MOFs used in removal of formaldehyde (FA).
3. p3_GAIresults.xlsx (prompt 3) contains the features responsible for FA removal in MOFs as generated by GAI models.
