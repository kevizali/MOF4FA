# Public repository for "AI-assisted screening of MOFs for formaldehyde removal: Combining GCMC simulation and machine learning"

Contents
1. p1_GAIresults.xlsx (prompt 1) contains the list of MOFs used in removal of VOCs generated by the GAI models.
2. p2_GAIresults.xlsx (prompt 2) contains the list of MOFs used in removal of formaldehyde (FA).
3. p3_GAIresults.xlsx (prompt 3) contains the features responsible for FA removal in MOFs as generated by GAI models.
4. svr_denoising.py contains the python script for dataset denoising.
5. mof4fa.py is the python script for training and testing FA adsoprion.
6. database.csv is the dataset for the selected 1208 CoRE MOFs.
7. denoised_data.csv is the SVR-based denoised dataset of database.csv.
8. hmof folder contains the predicted results of the seven ML models.
9. exp_mof.csv contains the SVR predicted FA adsorption values (FA_denoised_predicted) of HKUST-1 and MIL-101(Cr) MOFs.
